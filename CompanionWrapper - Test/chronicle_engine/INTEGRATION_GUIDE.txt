"""
SESSION CHRONICLE SYSTEM - Integration Guide

This document explains how to integrate the chronicle system into Kay's warmup flow.

═══════════════════════════════════════════════════════════
OVERVIEW
═══════════════════════════════════════════════════════════

The chronicle system replaces the multi-section warmup briefing with:
1. Kay's own essay from last session (PROMINENT)
2. Navigation tools (timeline, search, drill-down)
3. Scratchpad with session links
4. Current projects

═══════════════════════════════════════════════════════════
FILE STRUCTURE
═══════════════════════════════════════════════════════════

chronicle/
  session_chronicle.py    - Core data structure & storage
  chronicle_warmup.py     - Warmup briefing generator
  essay_prompt.py         - End-of-session essay system
  
data/
  session_chronicle.json  - Master timeline storage

═══════════════════════════════════════════════════════════
INTEGRATION POINTS
═══════════════════════════════════════════════════════════

1. KAY STARTUP (kay_ui.py)
   
   Add to imports:
   ```python
   from chronicle.session_chronicle import SessionChronicle, create_empty_chronicle
   from chronicle.chronicle_warmup import ChronicleWarmup
   ```
   
   In __init__:
   ```python
   # Initialize chronicle
   chronicle_path = self.base_dir / "data" / "session_chronicle.json"
   if not chronicle_path.exists():
       create_empty_chronicle(self.base_dir)
   self.chronicle = SessionChronicle(chronicle_path)
   
   # Create chronicle-based warmup
   self.chronicle_warmup = ChronicleWarmup(
       chronicle=self.chronicle,
       scratchpad=self.scratchpad,
       projects_tracker=self.projects_tracker  # You'll need to create this
   )
   ```

2. WARMUP GENERATION (warmup_engine.py)
   
   REPLACE the existing briefing generation with:
   ```python
   def generate_briefing(self):
       return self.chronicle_warmup.generate_briefing(
           current_session_order=self._current_session_order,
           current_time=datetime.now().isoformat()
       )
   ```

3. CHRONICLE COMMANDS (during warmup)
   
   In the warmup loop, handle chronicle navigation:
   ```python
   user_input = input("Kay> ").strip()
   
   # Chronicle commands
   if any(cmd in user_input.lower() for cmd in ["show timeline", "search chronicle", "detail session"]):
       response = self.chronicle_warmup.handle_chronicle_command(user_input)
       print(response)
       continue
   
   # Regular warmup commands
   if user_input.lower() == "ready":
       break
   ```

4. SESSION END (kay_ui.py)
   
   Add to imports:
   ```python
   from chronicle.essay_prompt import SessionEssayPrompt
   ```
   
   Create essay prompt system:
   ```python
   self.essay_prompt = SessionEssayPrompt(self.chronicle)
   ```
   
   When session ends:
   ```python
   def end_session(self):
       # Prompt Kay for essay
       prompt = self.essay_prompt.generate_essay_prompt(
           session_order=self._current_session_order,
           session_id=self.session_id,
           session_start_time=self.session_start_time,
           scratchpad_items_created=self.scratchpad.get_items_created_this_session(),
           scratchpad_items_resolved=self.scratchpad.get_items_resolved_this_session()
       )
       
       print(prompt)
       essay = self.get_kay_response("Write your essay for next-you:")
       
       # Save to chronicle
       self.essay_prompt.save_essay_to_chronicle(
           session_order=self._current_session_order,
           session_id=self.session_id,
           session_start_time=self.session_start_time,
           essay_text=essay,
           scratchpad_items_created=self.scratchpad.get_items_created_this_session(),
           scratchpad_items_resolved=self.scratchpad.get_items_resolved_this_session()
       )
   ```

═══════════════════════════════════════════════════════════
SCRATCHPAD INTEGRATION
═══════════════════════════════════════════════════════════

The scratchpad needs two new methods:

```python
def get_items_created_this_session(self) -> List[int]:
    \"\"\"Return IDs of items created during current session.\"\"\"
    # Track this during add_item()
    return self._session_created_items

def get_items_resolved_this_session(self) -> List[int]:
    \"\"\"Return IDs of items resolved during current session.\"\"\"
    # Track this during resolve_item()
    return self._session_resolved_items
```

Initialize these lists at session start and clear at session end.

═══════════════════════════════════════════════════════════
PROJECTS TRACKER
═══════════════════════════════════════════════════════════

Create a simple projects tracker (or integrate with existing system):

```python
class ProjectsTracker:
    def __init__(self, projects_file):
        self.projects_file = projects_file
        self.projects = self._load_projects()
    
    def _load_projects(self):
        if self.projects_file.exists():
            with open(self.projects_file, 'r') as f:
                return json.load(f)
        return {"active": []}
    
    def get_active_projects(self) -> List[str]:
        return self.projects.get("active", [])
    
    def add_project(self, name: str):
        if "active" not in self.projects:
            self.projects["active"] = []
        self.projects["active"].append(name)
        self._save()
    
    def _save(self):
        with open(self.projects_file, 'w') as f:
            json.dump(self.projects, f, indent=2)
```
