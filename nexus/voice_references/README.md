# Voice References for Voxtral TTS

Place voice reference WAV files here. The naming convention is:
- `kay_ref.wav` — Kay's voice reference (3-10 seconds of target voice)
- `reed_ref.wav` — Reed's voice reference

Voxtral can clone a voice from as little as 3 seconds of audio.
The reference should capture the target voice's natural rhythm, tone, and cadence.

## Tips for good references:
- 5-10 seconds is the sweet spot
- Clear audio, minimal background noise
- Natural speech (not reading stiffly)
- 24kHz WAV format preferred
- Mono channel

## Preset voices (if no reference file exists):
- Kay defaults to "casual_male"
- Reed defaults to "casual_female"
