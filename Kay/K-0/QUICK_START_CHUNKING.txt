STEP 3: NAVIGATE THROUGH DOCUMENT
================================================================================

Command                  Action
-----------------        -------------------------------------------------
"continue reading"       → Advance to next chunk (section 2/9, 3/9, etc.)
"next section"           → Same as continue reading
"previous section"       → Go back one chunk
"go back"                → Same as previous section
"jump to section 5"      → Jump directly to chunk 5
"restart document"       → Return to beginning (section 1/9)

Terminal shows new chunk immediately (no LLM call for navigation).

✓ "[DOC READER] Chunk added: ~25000 chars (section X/N)"

✓ Navigation commands work instantly
