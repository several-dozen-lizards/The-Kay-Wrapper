USER NAVIGATION COMMANDS
================================================================================

continue reading    → Advance to next section
next section        → Advance to next section
previous section    → Go back one section
go back             → Go back one section
jump to section N   → Jump to specific section (1-indexed)
restart document    → Return to beginning (section 1)
